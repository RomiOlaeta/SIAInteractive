# Proyecto Cypress - Dex Manager

## Descripción

Este proyecto contiene pruebas automatizadas End to End desarrolladas con Cypress para validar funcionalidades principales de la plataforma Dex Manager, como login, gestión de carpetas y carga de archivos en Media Library.

---

# Tecnologías utilizadas

- Node.js
- Cypress
- TypeScript
- Page Object Model (POM)
